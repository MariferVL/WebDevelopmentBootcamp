Position types

  1. Static: is by default.
  2. Relative:
    • allows you to change Coordinates(top, bottom, left, right)
    • The change doesn't affects/change others elements
  3. Absolute:
    • you give a margin to it's parent element.
    • So if you set a right:60px the element will move to the right side with a 60px margin
    • Affects the others elements as if the changed element was deleted
  4. Fixed:
    • The element it doesn't move from it's position.
